# Pawboo

반려묘와 반려견을 키우는 보호자(집사)들이 자신의 반려동물을 자랑하고, 일일 미션을 통해 특별한 순간을 기록하며, 커뮤니티 내에서 소통하는 **반려동물 특화 SNS 플랫폼**입니다.

## 핵심 가치

- **미션 기반 게이미피케이션**으로 지속적인 참여 유도
- **반려동물 캐릭터 중심**의 커뮤니티 문화 형성
- **사진 기록**을 통한 반려동물 라이프스타일 아카이빙

---

## 핵심 기능

| 기능 영역  | 기능명               | 설명                             |
|--------|-------------------|--------------------------------|
| 계정 관리  | 회원가입 / 로그인        | 소셜 로그인 (카카오)                   |
| 펫 캐릭터  | 펫 프로필 생성          | 반려동물 이름, 사진, 종류 등록             |
| 미션 시스템 | 일일 미션 제시 및 사진 업로드 | 매일 새로운 컨셉 미션 제공, 사진 업로드로 점수 획득 |
| 랭킹     | 주간/월간 랭킹 보드       | 미션 점수 합산 기준 랭킹 집계 및 공개         |
| 커뮤니티   | 펫 인스타그램형 프로필 & 피드 | 팔로우, 좋아요, 댓글, 미션 기록 피드         |

## 화면 구조 (IA)

하단 탭 네비게이션 기반으로 5개 주요 영역으로 구성됩니다.

| Depth 1 (탭) | Depth 2 (화면) | Depth 3 (세부 기능)                 |
|-------------|--------------|---------------------------------|
| **온보딩**     | 로그인 / 회원가입   | 카카오 소셜 로그인                      |
|             | 펫 등록 (최초)    | 펫 이름, 종류(고양이/강아지), 품종, 사진 업로드   |
| **홈 (피드)**  | 글로벌 피드       | 전체 유저 미션 완료 사진 피드, 좋아요/댓글       |
|             | 팔로잉 피드       | 팔로우한 펫들의 미션 사진 피드               |
| **미션**      | 오늘의 미션       | 일일 미션 확인, 가이드 예시 이미지, 포인트 안내    |
|             | 미션 업로드       | 사진 촬영 / 갤러리 선택, 필터, 코멘트 작성, 제출  |
|             | 미션 히스토리      | 내 펫의 과거 미션 완료 기록 타임라인           |
| **랭킹**      | 주간 랭킹        | 이번 주 점수 기준 전체 랭킹 TOP 100        |
|             | 월간 랭킹        | 이번 달 기준 종합 랭킹, 종류별 필터 (고양이/강아지) |
|             | 내 순위         | 내 펫의 현재 순위 및 주변 순위 확인           |
| **커뮤니티**    | 펫 프로필 탐색     | 다른 유저의 펫 프로필 방문, 팔로우            |
|             | 펫 프로필 페이지    | 인스타형 그리드, 미션 완료 사진 모음, 팔로워/팔로잉  |
| **마이페이지**   | 내 프로필        | 집사 정보, 내 펫 목록, 게시물 수, 팔로워       |
|             | 펫 프로필 관리     | 펫 정보 수정, 사진 변경, 펫 추가/삭제         |
|             | 설정           | 알림, 계정 관리, 개인정보 처리방침, 로그아웃      |

## 상세 기능 명세

### 회원/계정 관리 `AUTH-001 ~ AUTH-010`

| 항목       | 내용                                              |
|----------|-------------------------------------------------|
| 회원가입 방법  | 카카오 OAuth2.0                                    |
| 집사명(닉네임) | 2~20자, 특수문자 불가, 중복 불가, 추후 변경 가능 (30일 1회)        |
| 로그인 유지   | JWT Access Token (1시간) + Refresh Token (30일) 방식 |
| 회원 탈퇴    | 즉시 탈퇴 처리, 개인정보 30일 후 완전 삭제, 펫/게시물 익명 처리         |

### 펫 캐릭터 관리 `PET-001 ~ PET-015`

| 항목       | 내용                                                 |
|----------|----------------------------------------------------|
| 펫 등록 수   | 계정당 최대 5마리 등록 가능                                   |
| 등록 필수 정보 | 펫 이름(2~15자), 종류(고양이/강아지), 대표 사진 1장                 |
| 등록 선택 정보 | 품종, 생년월일, 성별, 한 줄 소개(최대 60자)                       |
| 대표 사진    | JPG/PNG/WEBP, 최대 10MB, 자동 리사이즈 (400×400px), CDN 저장 |
| 미션 활성 펫  | 미션 탭에서 미션 수행 시 '활성 펫' 1마리 선택 (전환 가능)               |
| 점수/랭킹 단위 | 펫 단위로 점수 관리 (유저가 여러 펫 보유 시 펫별 독립 랭킹)               |
| 펫 삭제     | 펫 삭제 시 해당 펫의 미션 기록/점수도 함께 삭제 (복구 불가)               |

### 일일 미션 시스템 `MISSION-001 ~ MISSION-020`

| 항목       | 내용                                                     |
|----------|--------------------------------------------------------|
| 미션 등록 주체 | DB 직접 삽입 (관리자 API는 추후 추가 예정)                           |
| 미션 활성화   | 매일 00:00 KST 자동 활성화 (날짜별 1개 보장, `scheduled_at` UNIQUE) |
| 참여 제한    | 펫 1마리당 하루 1회만 제출 가능, 수정 불가 (삭제 후 재제출 가능)               |
| 업로드 스펙   | 이미지: JPG/PNG/WEBP, 최대 10MB, 영상 업로드 미지원(v1)             |
| 이미지 처리   | 서버에서 리사이즈 (최대 1080px, inside fit), WebP 변환, CDN 저장     |
| 코멘트      | 미션 제출 시 코멘트 선택 입력 (최대 150자)                            |
| 해시태그     | # 태그 최대 5개, JSON 배열로 저장                                |
| 미션 삭제    | 삭제 가능, 삭제 시 획득 점수 차감 (재제출 가능)                          |
| 점수 단위    | 미션별 `baseScore` 기준, 삭제 시 차감 (최솟값 0)                    |

### 피드 및 게시물 `FEED-001 ~ FEED-015`

| 항목        | 내용                                 |
|-----------|------------------------------------|
| 피드 유형     | 글로벌 피드 (전체 공개) / 팔로잉 피드 (팔로우 대상만)  |
| 정렬 기준     | 글로벌: 최신순(기본), 인기순(좋아요 많은 순) 전환 가능  |
| 페이지네이션    | 무한 스크롤, 페이지당 20개 로드                |
| 좋아요       | 피드/상세 화면에서 하트 아이콘 탭, 중복 불가, 취소 가능  |
| 댓글        | 최대 300자, 대댓글 1 depth 지원, 신고 기능 포함  |
| 공유        | 인스타그램/카카오/링크 복사 공유 (해당 미션 게시물 URL) |
| 게시물 신고    | 부적절한 사진/스팸/저작권 위반 사유 선택 신고         |
| 게시물 공개 범위 | v1: 전체 공개만 지원 (추후 팔로워 공개 옵션 추가 예정) |

### 랭킹 시스템 `RANKING-001 ~ RANKING-010`

| 항목       | 내용                                    |
|----------|---------------------------------------|
| 랭킹 종류    | 주간 / 월간 / 종합(누적) / 종류별(고양이/강아지)       |
| 노출 범위    | TOP 100 노출, 본인 펫은 항상 하단에 고정 노출 (내 순위) |
| 실시간 반영   | 미션 완료/좋아요 수신 즉시 점수 반영, 랭킹은 5분 캐싱 후 갱신 |
| 랭킹 카드 정보 | 순위, 펫 프로필 사진, 펫 이름, 집사명, 총 점수, 뱃지     |
| TOP 3 처리 | 1/2/3위 별도 시각화 (금/은/동 배경), 왕관 아이콘      |

### 커뮤니티 (펫 프로필 / SNS 기능) `COMMUNITY-001 ~ COMMUNITY-015`

| 항목        | 내용                                       |
|-----------|------------------------------------------|
| 펫 프로필 페이지 | 대표 사진, 이름, 품종, 점수, 팔로워/팔로잉 수, 미션 그리드     |
| 그리드 구성    | 3열 인스타형 그리드, 최신 미션 사진 순, 탭 선택 시 상세 이동    |
| 팔로우       | 팔로우/언팔로우 토글, 팔로워/팔로잉 목록 조회 가능            |
| 검색 기능     | 펫 이름, 집사 닉네임, 해시태그 검색, 최근 검색어 저장(최대 10개) |
| 탐색 탭      | 인기 펫(팔로워 많은 순), 신규 펫, 미션 완료 많은 펫 큐레이션    |
| 프로필 꾸미기   | 한 줄 소개                                   |

## 개발 순서

### Phase 1 — 인증 / 계정

1. 카카오 OAuth2.0 로그인 / 회원가입
2. JWT Access Token + Refresh Token 발급 및 갱신
3. 로그아웃
4. 회원 탈퇴

### Phase 2 — 펫 캐릭터

1. 최초 펫 등록 (온보딩 플로우)
2. 펫 프로필 조회 / 수정 / 삭제
3. 펫 추가 등록 (최대 5마리)
4. 활성 펫 전환

### Phase 3 — 미션 시스템

1. 관리자 미션 등록 (제목, 설명, 예시 이미지, 예약 발행)
2. 오늘의 미션 + 활성 펫 제출 여부 조회
3. 미션 사진 제출 (이미지 리사이즈, CDN 저장, 점수 적립)
4. 미션 코멘트 / 해시태그 작성
5. 제출 취소 (점수 차감, 재제출 가능)
6. 미션 히스토리 조회 (cursor 페이지네이션)

### Phase 4 — 피드

1. 글로벌 피드 (최신순 / 인기순, 무한 스크롤)
2. 팔로잉 피드
3. 좋아요 / 취소
4. 댓글 / 대댓글
5. 게시물 신고
6. 공유 (링크 복사 / 카카오)

### Phase 5 — 랭킹

1. 주간 랭킹 집계 및 조회 (TOP 100)
2. 월간 랭킹
3. 종류별 필터 (고양이 / 강아지)
4. 내 순위 고정 노출

### Phase 6 — 커뮤니티 / SNS

1. 팔로우 / 언팔로우
2. 팔로워 / 팔로잉 목록 조회
3. 펫 프로필 페이지 (3열 그리드)
4. 검색 (펫 이름 / 닉네임 / 해시태그)
5. 탐색 탭 큐레이션
6. 프로필 꾸미기
